## ささかまぼこ 「言伝-Kotozute-」

## エレベーターピッチ
「この講義思ってたのと違う・・・」を解決したい
    筑波大生向けの感想機能つきシラバスのkotozuteです。
    これは授業に関するコメントや、満足度を入力・閲覧することができ、
    KdBと違ってシラバスにない情報まで知る事ができます。
